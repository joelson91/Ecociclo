# Ecociclo

O Ecociclo é um projeto voltado para a sustentabilidade, oferecendo aos usuários uma interface interativa para acompanhamento de metas, leitura de notícias e aprendizado.

## Pré-requisitos
- Python 3.7 ou superior
- Kivy 2.2.0 ou superior

## Instalação e Execução
1. Clone este repositório para o seu ambiente local.
```bash
git clone "https://github.com/joelson91/Ecociclo.git"
```

2. Crie e ative um ambiente virtual (recomendado).
```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Instale as dependências necessárias.
```bash
pip install -r requirements.txt
```

4. Execute o arquivo main.py na raiz do projeto.
```bash
python3 main.py
```
